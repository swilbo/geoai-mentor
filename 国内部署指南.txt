GeoAI Mentor — 国内平台部署指南
====================================

方案一：Gitee Pages（免费，推荐）
--------------------------------
1. 注册 https://gitee.com
2. 新建仓库（名称：geoai-mentor，选择"公开"）
3. 上传本文件夹中以下文件到仓库根目录：
   - index.html
   - mentor.html
   - resources.html
   - daily.html
   - 404.html
   - _next/ 文件夹（整个上传）
   - GeoAI-Mentor.html
   - 所有 .txt 文件
4. 仓库页面 → 服务 → Gitee Pages → 启动
5. 获得 URL：https://你的用户名.gitee.io/geoai-mentor

方案二：腾讯云 COS 静态托管
--------------------------------
1. 注册 https://cloud.tencent.com
2. 开通对象存储 COS
3. 创建存储桶（访问权限：公有读）
4. 开启静态网站功能
5. 上传本文件夹所有文件
6. 获得 URL

方案三：阿里云 OSS 静态托管
--------------------------------
1. 注册 https://www.aliyun.com
2. 开通对象存储 OSS
3. 创建 Bucket（公共读）
4. 设置静态页面默认首页为 index.html
5. 上传本文件夹所有文件
6. 可绑定自定义域名

方案四：Vercel / Netlify（境外，无需翻墙但需自定义域名）
--------------------------------
1. 将整个 haohao 文件夹初始化为 Git 仓库
2. 推送到 GitHub
3. 在 Vercel.com 导入 GitHub 仓库
4. 部署后在 Settings → Domains 绑定国内可访问的 .cn 域名

部署注意事项
--------------------------------
- 本网站为纯静态网站，不需要后端服务器
- DeepSeek API 在国内可直接访问，无需翻墙
- 如使用自有域名，需完成 ICP 备案
